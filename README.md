# 💳 BankLite – Console Banking System

BankLite is a simple, console-based banking simulation built using Python and object-oriented programming. Users can create accounts, deposit and withdraw money, view balances, and keep track of transaction history.

---

## 🧠 Features

- ✅ Create a bank account (auto-assigned ID)
- ✅ Deposit / Withdraw funds
- ✅ View balance
- ✅ View transaction history
- ✅ Save and load data using `bank.json`
